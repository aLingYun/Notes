## 题来了
设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。
实现 `MinStack` 类:
* `MinStack()` 初始化栈对象。
* `void push(int val)` 将元素val推入堆栈。
* `void pop()` 删除栈顶部的元素。
* `int top()` 获取栈顶部的元素。
* `int getMin()` 获取栈中的最小元素。
## 题解
本题解法的核心是除了存储数据的 DATA 栈，还需要额外一个最小栈。在进行数据压栈的时候，把最小的数值压入最小栈。
1. 往 data 栈中压入第一个元素时，同时也往最小栈中压入这个元素。
2. 后续再压入元素时（假定为 b），获取最小栈的栈顶元素（假定为a）与压入元素比较大小。并将元素 b 压入 data 栈，将 a 和 b 中较小的值压入最小栈。
3. 当数据栈往外弹出时，最小栈也跟着弹出即可。

## 完整代码
```rust
struct MinStack {
    deep: usize,
    data: Vec<i32>,
    min: Vec<i32>,
}

impl MinStack {
    fn new() -> Self {
        MinStack {
            deep: 0,
            data: vec![],
            min: vec![],
        }
    }

    fn push(&mut self, val: i32) {
        self.data.push(val);
        //将较小的值压入最小栈
        if self.deep == 0 || self.get_min() > val {
            self.min.push(val);   
        } else {
            self.min.push(self.get_min());
        }
        self.deep += 1;
    }

    fn pop(&mut self) {
        self.deep -= 1;
        self.min.pop();
        self.data.pop();
    }

    fn top(&self) -> i32 {
        self.data[self.deep - 1]
    }

    fn get_min(&self) -> i32 {
        self.min[self.deep - 1]
    }
}

fn main() {
    let mut min_stack = MinStack::new();
    min_stack.push(3);
    println!("min = {}", min_stack.get_min());
    min_stack.push(-1);
    println!("min = {}", min_stack.get_min());
    min_stack.push(0);
    println!("min = {}", min_stack.get_min());
    println!("top = {}", min_stack.top());
    min_stack.pop();
    println!("min = {}", min_stack.get_min());
    min_stack.pop();
    println!("min = {}", min_stack.get_min());
    println!("Hello, world!");
}
```
